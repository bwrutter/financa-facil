# Finança Fácil API (Em desenvolvimento, vai ser uma aplicação completa)

API para gerenciamento de finanças pessoais, permitindo controlar usuários, contas e transações.  
Utiliza **Node.js**, **Express**, **MongoDB (Atlas)** e **JWT** para autenticação.

## Tecnologias

- Node.js v22+
- Express
- MongoDB (Atlas)
- Mongoose
- JWT (Json Web Token)
- bcryptjs
- Swagger (Documentação)
- Nodemon (desenvolvimento)
